# Delete a Scout account

This page is ready to adapt for Scout's future public support site.

1. Open the Scout extension.
2. Open **Account**.
3. Scroll to **Your data**.
4. Select **Delete account**.
5. Review the deletion warning and confirm.

Scout requests deletion of the signed-in authentication identity, private
database records, uploaded resume files, sessions, and that account's cache on
the current browser. This action cannot be undone.

If the extension is unavailable, contact **[SUPPORT EMAIL]** from the email
address associated with the Scout account. Scout may ask you to complete an
account-verification step before processing the request.

Private support reports are normally retained for up to 90 days. Infrastructure
backups may persist temporarily according to the provider's backup lifecycle and
are not used to recreate a deleted user account.
